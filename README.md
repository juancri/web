# Original repo

https://github.com/11ty/eleventy-base-blog.git

# Dependencies

Requires:

- Node 16.x

# Install packages

```
npm install
```

# Run Eleventy

```
npx eleventy
```

Or build and host locally for local development
```
npx eleventy --serve
```

Or build automatically when a template changes:
```
npx eleventy --watch
```

Or in debug mode:
```
DEBUG=* npx eleventy
```
