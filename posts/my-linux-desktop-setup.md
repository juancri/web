# Introduction

I've been using Linux for more than 20 years now. It took me several years to make Linux my primary desktop driver, though. Here's a snapshot of my current setup, from the distro to the tools I use for software development.

## The distro

My first distro was SuSE 5.1. After a while, I used RedHat, Debian, Gentoo and Ubuntu.

It was not easy to use Linux during the late 90s and early 2000s as my primary desktop environment. When laptops stated to be a thing, simple things like connecting to a new wireless network was a not-easy task on Linux. Ubuntu changed that and it was really thought as a nice desktop experience. That was the point when I decided to fully move from Windows to Linux.

After Ubuntu, I jumped to Arch and I currently use several variants of it, but mainly vanilla Arch.

## The desktop

My favorite desktop environment is MATE (pronounced *mate* as the drink in Spanish). It might be that I'm just old and I miss the early 2000s, but MATE + compiz are usually the base I choose for my desktop.

I use MATE but that does not mean that I don't like modern desktop experiences. That's why I use synapse and plank as the launcher and dock, respectively.
